Here are the things that I use to make The Drum Thing streams timestamps or Tim Tams.
Everything here was made with the help of locally run AI (Qwen3.6 at the moment).

The main site is here: [https://mario-marin.github.io/goob_tool/](https://mario-marin.github.io/goob_tool/)

And you can view all the timestamps with all the fun statistics! More will probably come in the future.
If you find any mistakes, fixes or just want to say Hi you can leave an issue on this repo!

There's not much here but this is what you can find:
* tim-tams-viewer: Thats the site, the data is stored on the public/data folder!
* timestamp_tool.py: That0s the main tool to make the Tim Tams, the Tim Tams bakery if you will! I add features to it as I need.
* json_convert_timestamps.py: This is our little fella that converts the Tim Tams into a json for the site to read.
* update_tracks_stats.py: Updates the tracks statistics that are shown on the site.
* generate-timtams.sh: A little script to generate everything and deploy it, very handy.
